## SCAN系プロンプト（#01〜#10）

### #01 マクロ環境総合スキャン

**用途**: 毎週月曜日に実行。市場全体の方向性を把握する。

```
あなたは経験豊富なマクロ経済アナリストです。

現在のグローバル経済および日本市場について、
以下の構造で分析を整理してください。

【1. 金融政策】
- 日銀の直近の政策とスタンス
- FRBの直近の政策とスタンス
- 今後3ヶ月の政策変更見通し

【2. 為替動向】
- ドル円の直近トレンドと主な要因
- 為替が日本株に与える影響の方向性

【3. 地政学リスク】
- 現在注目すべき地政学的リスク（上位3つ）
- 各リスクの市場影響度（高/中/低）

【4. 市場環境指標】
- VIX（恐怖指数）の水準感
- 日経平均のテクニカル的な位置

【5. 総合評価】
- 市場環境を5段階で評価（1=非常に弱気 〜 5=非常に強気）
- 評価の根拠を3行で

※ 不確実な情報には「推定」と明記してください。
```

**読み方**: 総合評価3以上→攻勢、3未満→守勢。地政学リスク「高」がある場合はポジション縮小を検討。

### #11 投資判断の問い定義

**用途**: 判断を始める前の言語化。

```
以下の投資アイデアを、判断可能な「問い」に変換してください。

【投資アイデア（ぼんやりした段階でOK）】
（※ 「テスラが気になる」「半導体に投資したい」等）

【変換後の問い（以下の要素を含めて）】
- 対象: 何に投資するか
- 金額: いくら投入するか
- 期間: いつまで保有するか
- 目的: 何を期待するか（成長/配当/分散等）
- アクション: 買う/追加/持続/売却のどれか
```

### #12 多角分析リクエスト

**用途**: 投資判断を多面的に分析する。Step 2で使用。

```
以下の投資判断について、4つの観点から分析してください。

【判断】（※ #11の出力を貼り付け）

1. ファンダメンタルズ（財務・成長性・バリュエーション）

3. マクロ環境との整合性
4. ポートフォリオ観点（分散・集中度）

各観点を5段階評価 + 根拠で出力。

最後に：投資を支持する根拠3つ / 反対する根拠3つ
```

## プロンプトのカスタマイズガイド

### 基本原則
1. **構造を変えず、対象を変える** — フォーマットはそのまま、銘柄名や条件を差し替える
2. **出力粒度を調整する** — 「3つ」を「5つ」に増やしたり、「3行」を「5行」に増やしたり
3. **ペルソナを追加する** — 冒頭に「あなたは〇〇の専門家です」を追加して回答の角度を変える

### AIの選び方
- **深い分析が必要な場合**: Claude（長文の構造化に強い）
- **最新情報が必要な場合**: ChatGPT（検索機能付きモデル）、Perplexity
- **複数の視点が欲しい場合**: 2〜3種類のAIで同じプロンプトを実行して比較
