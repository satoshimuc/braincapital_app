## SCANフレームワークとは

SCANは「情報を集める」行為ではない。**「情報を構造化して俯瞰する」**行為だ。

従来の情報収集は、ニュースを読み、レポートを読み、SNSをチェックし、それらを頭の中でつなぎ合わせる手作業だった。これには時間がかかる上に、見落としやバイアスが避けられない。

SCANフレームワークでは、AIを使って4つのレイヤーを**同時並行**でスキャンする。手作業で3〜5時間かかる情報整理を、15〜30分に圧縮する。

```
┌────────────────────────────────────────┐
│         SCANフレームワーク 4レイヤー      │
├────────────────────────────────────────┤
│                                        │
│  Layer 1: マクロスキャン                │
│  → 市場全体の潮流を俯瞰する            │
│                 ↓                      │
│  Layer 2: セクタースキャン              │
│  → 業界別の動向を比較する              │
│                 ↓                      │
│  Layer 3: 個別スキャン                  │
│  → 銘柄レベルの分析を構造化する        │
│                 ↓                      │
│  Layer 4: センチメントスキャン          │
│  → 市場心理の温度を測る                │
│                                        │
└────────────────────────────────────────┘
```

上から下に向かって、視野が「広い→狭い」に絞られていく。この順序が重要だ。マクロを見ずに個別銘柄を分析するのは、地図を見ずに道を歩くようなものだ。

## Layer 1: マクロスキャン — 市場全体の潮流を俯瞰する

### 目的

世界経済および日本市場全体の方向性を掴む。金利、為替、地政学リスク、政策動向など、個別銘柄の上位に位置するマクロ要因を構造化する。

### なぜマクロから始めるのか

個別銘柄がどれだけ好業績でも、マクロ環境が逆風なら株価は下がる。逆に、マクロが追い風なら平凡な銘柄でも上昇する。投資判断の精度は、マクロ認識の精度で決まる。

### プロンプト #1: マクロ環境総合スキャン

```
あなたは経験豊富なマクロ経済アナリストです。

現在のグローバル経済および日本市場について、
以下の構造で分析を整理してください。

【1. 金融政策】
- 日銀の直近の政策とスタンス
- FRBの直近の政策とスタンス
- 今後3ヶ月の政策変更見通し

【2. 為替動向】
- ドル円の直近トレンドと主な要因
- 為替が日本株に与える影響の方向性

【3. 地政学リスク】
- 現在注目すべき地政学的リスク（上位3つ）
- 各リスクの市場影響度（高/中/低）

【4. 市場センチメント指標】
- VIX（恐怖指数）の水準感
- 日経平均のテクニカル的な位置

【5. 総合評価】
- 市場環境を5段階で評価（1=非常に弱気 〜 5=非常に強気）
- 評価の根拠を3行で

※ 不確実な情報には必ず「推定」と明記してください。
※ データの時点を明記してください。
```

### 出力の読み方

このプロンプトの出力で特に注目すべきポイント：

- **金融政策の方向性** — 利上げ/利下げのどちらに向かっているか。これが株式市場全体の方向性を決める最大の要因
- **総合評価のスコア** — 3以上なら攻め（成長株重視）、3未満なら守り（ディフェンシブ重視）の基本スタンスを取る
- **地政学リスクの影響度「高」** — これがある場合は、ポジションサイズを通常の70〜80%に抑制する

### プロンプト #2: 前週比マクロ変化スキャン

```
先週の主要マクロ経済イベントおよびデータについて、
以下の構造で変化点を整理してください。

【前週からの主要変化】（箇条書き5つ以内）

【変化の影響分析】
| 変化要因 | 影響を受けるセクター | 影響の方向（+/-） | 影響度 |
|----------|---------------------|------------------|--------|

【今週の注目イベント・指標発表】
- 日付、イベント名、注目理由を一覧で

【アクション示唆】
- 上記を踏まえて、投資家が今週注意すべきこと（3つ）
```

このプロンプトは**毎週月曜日に実行**する。前回のSCAN結果と比較することで、変化の方向とスピードが見える。

## Layer 2: セクタースキャン — 業界別の動向を比較する

### 目的

マクロ環境を踏まえた上で、どのセクターに追い風が吹いているか、どのセクターが逆風を受けているかを比較分析する。

### なぜセクター単位で見るのか

個別銘柄のパフォーマンスの約60〜70%は、そのセクター全体の動向で説明できるという研究がある。つまり、良いセクターの中から銘柄を選ぶのが、良い銘柄を全セクターから探すよりも効率が良い。

### プロンプト #3: セクター比較分析

```
日本株式市場の主要セクターについて、
以下の構造で比較分析してください。

【対象セクター】
半導体/AI関連、自動車、銀行・金融、医薬品・ヘルスケア、
エネルギー、不動産、小売・消費財、通信・IT

【各セクターの分析項目】
1. 直近3ヶ月のトレンド（上昇/横ばい/下降）
2. 主な推進要因（追い風要因 2つ）
3. 主なリスク要因（逆風要因 2つ）
4. 注目すべき先行指標
5. 総合魅力度スコア（5段階）

【出力フォーマット】
セクターごとに上記項目を表形式で整理してください。

【最後に】
- 現時点で最も魅力的なセクター上位3つとその理由
- 現時点で最も注意すべきセクター下位2つとその理由
```

### 出力の読み方

- **魅力度スコア4〜5のセクター** — ここから個別銘柄を探す（Layer 3へ進む対象）
- **スコア1〜2のセクター** — 保有中の銘柄があれば、ポジション縮小を検討
- **追い風/逆風のバランス** — 追い風だけのセクターは過熱気味の可能性。逆風がありながらスコアが高いセクターが実は狙い目

### プロンプト #4: セクターローテーション分析

```
現在の経済サイクル（回復期/拡大期/後退期/縮小期）を
あなたの分析に基づいて判定し、
以下の構造でセクターローテーション分析を行ってください。

【経済サイクルの現在地】
- 現在のフェーズとその根拠

【各フェーズで有利なセクター】
| フェーズ | 有利なセクター | 不利なセクター |
|----------|---------------|---------------|

【現在のフェーズで取るべきセクター配分】
- 推奨オーバーウェイト（3セクター）
- 推奨アンダーウェイト（3セクター）
- 各々の理由
```

## Layer 4: センチメントスキャン — 市場心理の温度を測る

### 目的

数値データだけでは捉えられない「市場の空気」を構造化する。過度な楽観も悲観も、逆張りのシグナルになり得る。

### プロンプト #8: 市場センチメント分析

```
現在の日本株式市場のセンチメント（投資家心理）について、
以下の構造で分析してください。

【センチメント指標】
| 指標 | 現在の水準 | 示唆 |
|------|-----------|------|
| VIX | | |
| 騰落レシオ | | |
| 信用買い残 | | |
| 投資主体別動向 | | |
| SNS/ニュースのトーン | | |

【総合センチメント評価】
- スケール: 1(極度の恐怖) 〜 10(極度の欲望)
- 現在のスコアと根拠

【逆張りシグナル】
- 現在のセンチメントから示唆される逆張りの機会はあるか
- ある場合、具体的にどのような行動が考えられるか

【センチメントの変化方向】
- 1ヶ月前と比較して、楽観/悲観どちらに傾いているか
- その変化のトリガーは何か
```

### プロンプト #9: ニュースセンチメント解析

```
以下のニュースヘッドライン群について、
市場センチメントの観点から分析してください。

【ニュース一覧】
（※ ここに直近1週間の主要ニュースヘッドラインを貼り付ける）

【分析項目】
1. 各ニュースのセンチメント分類（ポジティブ/ネガティブ/中立）
2. 全体的なトーン（楽観/悲観/混在）
3. 最も市場に影響を与えそうなニュース上位3つ
4. 報道の偏り（過度に楽観的/過度に悲観的な報道はないか）
5. 「報道されていないが重要な視点」の指摘
```

## SCANの実践ワークフロー

### 週次SCANルーティン

SCANは毎週1回、決まった曜日に実行する。推奨は**日曜夜または月曜朝**。

```
【週次SCANスケジュール】

Step 1: マクロスキャン（10分）
  → プロンプト#1 + #2 を実行
  → 市場全体の方向性を把握

Step 2: セクタースキャン（10分）
  → プロンプト#3 を実行
  → 注目セクターを特定

Step 3: 個別スキャン（15分）
  → プロンプト#5 を注目銘柄に対して実行（2〜3銘柄）
  → 保有銘柄の変化確認

Step 4: センチメントスキャン（5分）
  → プロンプト#8 を実行
  → 市場の温度感を確認

合計所要時間: 約40分
```

### 結果の保存・蓄積方法

SCANの結果は必ず保存する。推奨はスプレッドシートまたはNotionでの管理。

```
【SCAN結果 記録テンプレート】

日付: 20XX/XX/XX
━━━━━━━━━━━━━━━━━━━━
■ マクロ環境スコア: /5
  主な変化:

  1位:        （スコア: /5）

  3位:        （スコア: /5）

■ 注目銘柄:
  銘柄A:      （スコア: /10）

■ センチメント: /10
```

3週間分以上蓄積すると、トレンドの変化が見えてくる。スコアの推移をグラフ化すれば、さらに視覚的に把握できる。
