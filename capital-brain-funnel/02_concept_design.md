## 2. ブランドアイデンティティ

### コアバリュー
```
Capital（資本）× Brain（知性）× AI（人工知能）
= AIを活用して知性を資本化するフレームワーク
```

### ブランドパーソナリティ
- **知的**: データと論理に基づく発信
- **先進的**: AI活用の最前線
- **実践的**: 理論だけでなく具体的アクション
- **クール**: 煽りではなく冷静な分析

| 文体 | 断定的・簡潔・データドリブン |

## 3. コンテンツ体系（知識アーキテクチャ）

### Capital Brain 5つの柱
```
┌─────────────────────────────────────────┐
│           Capital Brain Framework        │
├─────────┬─────────┬─────────┬──────────┤
│  SCAN   │  THINK  │  BUILD  │  SCALE   │
│ 情報収集 │  思考法  │ 構築術  │  拡張    │
│         │         │         │          │
│ AI情報  │ 投資    │ ポート  │ 自動化   │
│ リサーチ │ フレーム │ フォリオ │ システム │
│ 術     │ ワーク  │ 設計   │ 構築    │
├─────────┴─────────┴─────────┴──────────┤
│              MINDSET（土台）              │
│        資本思考マインドセット              │
└─────────────────────────────────────────┘
```

### 各柱の詳細

#### Pillar 1: SCAN（スキャン）- AI情報リサーチ術
- AIで市場情報を高速収集する方法
- ノイズを除去し本質を抽出するプロンプト設計
- 情報ソースの格付けフレームワーク

#### Pillar 2: THINK（シンク）- 投資フレームワーク
- AIを使った企業分析・銘柄スクリーニング
- リスク評価マトリクスの構築法
- 意思決定バイアスの排除法

#### Pillar 3: BUILD（ビルド）- ポートフォリオ構築術
- AI支援ポートフォリオ設計
- セクター分析とアセットアロケーション
- リバランスの自動化戦略

#### Pillar 4: SCALE（スケール）- 自動化・拡張
- 投資判断の半自動化ワークフロー
- AIアラートシステムの構築
- 複数収入源への拡張戦略

#### Pillar 5: MINDSET（マインドセット）- 資本思考の土台
- 「消費脳」から「投資脳」への転換
- 長期思考と複利の本質
- 感情排除の意思決定フレームワーク
