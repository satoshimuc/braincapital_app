## 4. 導線の最適化施策

### A/Bテスト項目

| テスト対象 | パターンA | パターンB |
|-----------|----------|----------|
| Xプロフリンク | note直リンク | lit.link経由 |
| note CTA | 有料記事のみ案内 | 有料記事 + LINE同時案内 |
| 無料記事の数 | 3本 | 5本（追加2本で信頼構築強化） |
| 有料ラインの位置 | 全体の18% | 全体の25% |
| LINE特典 | 診断キットのみ | 診断キット + ミニ講座 |

### 分析ツール

| 用途 | ツール |
|------|--------|
| X分析 | Xアナリティクス |
| note分析 | noteダッシュボード |
| LINE分析 | LINE公式管理画面 |
| リンク計測 | lit.link / Bitly |
| 全体管理 | Notion / スプレッドシート |

## 6. リスク管理

| リスク | 対策 |
|--------|------|
| noteプラットフォーム依存 | LINE/メルマガで自社リスト構築 |
| X アカウント凍結 | 規約遵守・複数チャネル運用 |
| WEFレポートの誤引用 | 原文を正確に確認・引用元を明記 |
| エセ脳科学との混同 | 論文引用・エビデンスベースを徹底 |
| 競合の参入 | 先行者優位の確立・コミュニティでの差別化 |
| 医療行為との境界 | 「情報提供」に限定、治療助言はしない |
| コンテンツのネタ切れ | 3本柱 × WEF 5レバー × 最新論文で無限展開 |
