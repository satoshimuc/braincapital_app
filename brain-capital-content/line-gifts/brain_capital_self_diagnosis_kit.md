# Brain Capital セルフ診断キット
## Brain Health（15項目）+ Brain Skills（15項目）自己評価

**LINE登録者限定 無料特典**

---

あなたの「脳の資産」は今、どんな状態だろうか？

金融資産なら残高を見れば分かる。しかし**脳の資産（Brain Capital）**は、意識的に「測定」しなければ現状が見えない。

このセルフ診断キットは、世界経済フォーラム（WEF）× McKinseyレポート「The Human Advantage」のフレームワークに基づき、**Brain Health（15項目）+ Brain Skills（15項目）= 計30項目**であなたのBrain Capitalを可視化する。

所要時間は約10分。
月に1回実施して変化をトラッキングすることを推奨する。

---

## 使い方

各項目を **1〜5点** で自己評価する。

| スコア | 基準 |
|--------|------|
| 1 | まったく当てはまらない |
| 2 | あまり当てはまらない |
| 3 | どちらとも言えない |
| 4 | やや当てはまる |
| 5 | 非常に当てはまる |

正直に、直感で答えること。「こうありたい」ではなく「今の自分」を評価する。

# Part 1: Brain Health（脳の健康）— 15項目

Brain Healthは、Brain Skillsを発揮するための**土台**だ。
脳が疲弊していたら、どんなスキルも機能しない。

## カテゴリA: 睡眠と回復（3項目）

脳のメンテナンスシステム（グリンパティックシステム）は睡眠中に最も活発に機能する。睡眠の質は脳のパフォーマンスの最大の規定因子。

```
BH-01: 毎日7〜9時間の睡眠を確保できている
       1 ─ 2 ─ 3 ─ 4 ─ 5    →  [    ]点

BH-02: 朝の起床時にすっきりした状態で目覚められる
       1 ─ 2 ─ 3 ─ 4 ─ 5    →  [    ]点

BH-03: 日中に強い眠気を感じることなく過ごせている
       1 ─ 2 ─ 3 ─ 4 ─ 5    →  [    ]点
```

# Part 2: Brain Skills（脳のスキル）— 15項目

Brain Skillsは、AI時代に人間が価値を生み出すための**競争優位性**だ。
WEFが定義する5つのスキル領域について、各3項目で評価する。

## カテゴリF: 認知力 — Cognitive Skills（3項目）

批判的思考、論理的推論、問題解決。AIの出力を正しく評価し、意思決定に活かす力。

```
BS-01: 情報やニュースを見たとき「本当にそうか？」と立ち止まって考える習慣がある
       1 ─ 2 ─ 3 ─ 4 ─ 5    →  [    ]点

BS-02: 複雑な問題を構造化し、要因を分解して考えることができる
       1 ─ 2 ─ 3 ─ 4 ─ 5    →  [    ]点

BS-03: AIの出力に対して「前提は正しいか」「別の解釈はないか」を検証できる
       1 ─ 2 ─ 3 ─ 4 ─ 5    →  [    ]点
```

**小計F：[    ] / 15点**

## スコア解釈ガイド

### 総合スコア（150点満点）

| スコア帯 | レベル | 状態 |
|----------|--------|------|
| **121〜150** | S: Elite | 脳の資産が十分に蓄積されている。維持とさらなる成長を |
| **91〜120** | A: Strong | 良好な状態。弱い領域を特定し、戦略的に強化すれば大幅に伸びる |
| **61〜90** | B: Developing | 伸びしろが大きい。優先順位をつけて改善に取り組むフェーズ |
| **31〜60** | C: At Risk | 脳の資本が消耗している可能性。まずBrain Healthの立て直しが最優先 |
| **〜30** | D: Critical | 早急な対策が必要。睡眠・ストレス管理から着手を |

### Brain Health と Brain Skills のバランス

```
            Brain Skills スコア
                  高い
                   │
        ┌──────────┼──────────┐
        │          │          │
        │  Type B  │  Type S  │
        │ スキル偏重│  理想型  │
        │          │          │
Brain ──┼──────────┼──────────┼── Brain
Health   │          │          │   Health
低い    │  Type D  │  Type A  │   高い
        │ 要改善  │ 土台充実 │
        │          │          │
        └──────────┼──────────┘
                   │
                  低い
            Brain Skills スコア
```

**Type S（理想型）:** Health・Skills共に高い。脳の複利効果が最大に機能している状態。

**Type A（土台充実型）:** Brain Healthは整っているが、Brain Skillsの開発余地がある。脳のコンディションは良いので、スキルトレーニングに投資すれば大きく伸びる。

**Type B（スキル偏重型）:** Brain Skillsは高いが、Brain Healthが低い。ハイパフォーマンスだが持続性にリスクがある。バーンアウト予備軍の可能性。**Brain Healthの立て直しが急務。**

**Type D（要改善型）:** 両方が低い状態。まずBrain Health（睡眠・運動・ストレス管理）から着手する。Health無くしてSkillsの向上は不可能。
